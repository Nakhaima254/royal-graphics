# Royal Graphics Solutions

Premium Digital Marketing Agency specializing in graphic design, SEO services, and social media management.

## Project Overview

This is a React-based website for Royal Graphics Solutions, a digital marketing agency offering:
- Graphic Design Services
- SEO (Search Engine Optimization)
- Social Media Management
- Brand Identity Development

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/       # React components
│   ├── ui/          # shadcn/ui components
│   └── *.tsx       # Custom components
├── pages/           # Page components
├── assets/          # Images and static files
├── App.tsx         # Main app component
└── main.tsx        # Entry point
```

## Features

- Responsive design for all screen sizes
- Typewriter animation on hero section
- Animated counters
- Service testimonials carousel
- Contact form with validation
- SEO optimized
